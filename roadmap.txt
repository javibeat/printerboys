# Roadmap para Web Catálogo de Productos Impresos en 3D

## 1. Definición y Diseño Inicial
- [x] Definir branding inicial: paleta de colores (azul Apple #0071e3, blanco #ffffff, naranja acento #ff9500)
- [x] Selección de tipografías: Montserrat, Clash Grotesk, Bebas Neue
- [ ] Bocetos wireframe (desktop, tablet, móvil)
- [ ] Definir estructura de navegación y arquitectura de la información

## 2. Configuración del Proyecto
- [x] Inicializar proyecto Next.js con TypeScript
- [x] Instalar y configurar ESLint y Prettier
- [x] Instalar y configurar TailwindCSS
- [x] Organizar fuentes en /public/fonts y configurar en Tailwind
- [x] Configurar estructura de carpetas (/components, /pages, /styles, /utils, etc.)
- [x] Configuración de git, ramas (main/dev) y colaboración en GitHub

## 3. Maquetación y Componentes Base
- [x] Layout general (estructura base)
- [x] Header minimalista (logo, navegación, selector de idioma)
- [x] Footer minimalista (info básica, enlaces)
- [x] Home minimalista con hero section estilo Bento
- [ ] Grid de productos (tarjetas con imagen, nombre, descripción, precio y enlace externo)
- [ ] Página de contacto o sección de contacto

## 4. Responsividad y Accesibilidad
- [ ] Mobile-first, breakpoints para tablet y desktop
- [ ] Accesibilidad (WCAG, navegación teclado, contraste, etiquetas ARIA)

## 5. SEO y Performance
- [ ] Estructura semántica (HTML5, headings, landmarks)
- [ ] Meta tags, Open Graph, sitemap.xml, robots.txt
- [ ] Optimización de imágenes (WebP, lazy loading)
- [ ] Core Web Vitals (Lighthouse)

## 6. Funcionalidad y Experiencia de Usuario
- [ ] Animaciones suaves y microinteracciones (framer-motion, CSS transitions)
- [ ] Soporte multilenguaje (inglés y español)

## 7. Despliegue y Analítica
- [ ] Deploy en Vercel/Netlify
- [ ] Configurar Google Analytics y/o Plausible
- [ ] Monitorización de errores (Sentry)

## 8. Iteración y Feedback
- [ ] Testeo con usuarios reales
- [ ] Mejoras continuas según feedback

---

**Notas:**
- Web tipo escaparate/catálogo, sin registro ni compras internas.
- Cada producto enlaza a Etsy u otra plataforma externa.
- Mantener siempre el diseño minimalista y elegante.
- Priorizar la experiencia móvil y la velocidad de carga.
- Documentar cada paso y decisión relevante.
